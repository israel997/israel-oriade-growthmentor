# PWA Icons

Place the following icons here:
- `icon-192x192.png` — Standard PWA icon (192×192)
- `icon-512x512.png` — Standard PWA icon (512×512)
- `icon-maskable-512x512.png` — Maskable icon for Android adaptive icons (512×512, safe zone: inner 80%)

Generate them from your logo using https://maskable.app or https://realfavicongenerator.net
